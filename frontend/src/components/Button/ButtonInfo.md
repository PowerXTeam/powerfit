# Available classes

If there is icon, the button is automatically assigned the class _"button-w-icon"_.
___

The button has the following props:

1. type = "button-succes",
2. type = "button-danger",
3. type = "button-info".

___

There is also an optional _"isDisabled"_ props that deactivates the button.

___

The button has a required parameter content.
